# Bachelorproef

> Wanneer is EventSourcing een meerwaarde voor een bedrijf zoals Skedify, waar ze gespecialiseerd zijn in online scheduling?

## Installation

- `git clone git@github.com:RobinMalfait/bachelorproef.git bp-Robin-Malfait`
- `cd bp-Robin-Malfait`
- `npm install`

## Usage

### Build

- `npm run build`

### Cleanup 

- `npm run cleanup`

## Validate text

- `npm run validate`

## Wordcount

- `npm run wordcount`
